# Evidence Strategy

Status: draft

## Purpose

Define how evidence should be gathered for the Unified Tier Framework, wickedness scoring, readiness, BKI, and any future layered scoring systems.

## Core Evidence Types

- literature evidence
- rubric evidence
- classifier evidence
- case evidence
- operational evidence
- negative evidence

## Evidence Questions

1. Does the manual rubric produce consistent human judgments?
2. Does the classifier approximate the rubric well enough to be useful?
3. Does the tier or profile improve Base-N selection?
4. Does adding readiness or BKI explain outcomes better than wickedness alone?

## Required Artifacts

- goldset cases
- scoring sheets
- classifier comparisons
- case logs
- failure logs

## Claim Discipline

Do not label a framework element as validated unless the evidence type is named explicitly:

- theory-backed
- manually reliable
- classifier-supported
- outcome-supported
