# Bio-Cognitive Protocol Hints

Status: draft

## Purpose

Provide a lightweight bridge from the framework to the first bio-cognitive empirical protocols.

## Why This Matters

The framework now recognizes that bio-cognitive operating condition modifies readiness and BKI.

That needs empirical hooks, not only conceptual language.

## Suggested Uses

When a case involves:

- long-duration work
- sustained coordination
- overload risk
- degraded attention
- trace usability under fatigue

consider logging:

- readiness effects
- BKI effects
- bio-cognitive observations

## Example Questions

- did overload reduce execution capacity?
- did fatigue reduce willingness to dissent?
- did strain degrade shared interpretation quality?
- did a reversible adaptation improve readiness or BKI signals?

## Working Rule

Use bio-cognitive notes as a modifier on readiness and BKI, not as a fourth top-level framework layer.
