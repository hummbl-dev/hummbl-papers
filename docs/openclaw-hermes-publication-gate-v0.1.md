## Candidate report thesis

The working thesis to test is that OpenClaw and Hermes converged on overlapping agent-platform capabilities through opposite developmental paths:

- OpenClaw: messaging assistant → Gateway/control plane → multi-agent platform
- Hermes: research/trajectory harness → coding agent → persistent learning runtime → multi-surface platform

The report must evaluate whether this path dependence still shapes architecture, governance, and role fitness after feature convergence.
